# AI Data Analyst Tool

## Overview

AI Data Analyst Tool is a Django-based web application that helps users upload datasets, perform automated data analysis, generate visualizations, and obtain meaningful insights from data.

## Features

* Upload CSV datasets
* Automatic data analysis
* Dataset summary statistics
* Missing value detection
* Interactive charts and visualizations
* Data insights and reporting
* User-friendly dashboard
* Responsive interface

## Technologies Used

* Python
* Django
* Pandas
* NumPy
* Matplotlib
* Plotly
* HTML
* CSS
* JavaScript

### Navigate to Project Folder

cd ai-data-analyst-tool

### Create Virtual Environment

python -m venv venv

### Activate Virtual Environment

Windows:
venv\Scripts\activate

### Install Dependencies

pip install -r requirements.txt

### Run Migrations

python manage.py migrate

### Start Server

python manage.py runserver

## Usage

1. Open the application in your browser.
2. Upload a CSV dataset.
3. View dataset statistics and analysis.
4. Explore charts and visualizations.
5. Generate insights from the uploaded data.
